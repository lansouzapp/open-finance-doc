A Política de Versionamento do Open Finance Brasil (OFB) foi escrita para assegurar que os participantes do ecossistema tenham clareza sobre quais aspectos são necessários para o desenvolvimento de uma API no padrão OFB. Espera-se que através dela fiquem estabelecidas, de maneira clara aos participantes, as regras para o lançamento de novas versões das APIs existentes, o ciclo de vida de uma API, os prazos de retorno do ecossistema quanto a ajustes nas versões release candidates, os prazos de convivência das diferentes versões destas APIs e ainda orienta quais versões os novos participantes do ecossistema devem implementar.
