1. [ RECEIVED ] = Momento que credora recebe solicitação de portabilidade de crédito, este estado deve permanecer até o próximo dia útil (D+1);
2. [ PENDING ] = Prazo de até 3 dias úteis para a etapa de contraproposta, situações podem ocorrer neste estado: Cliente cancelar a solicitação de portabilidade(via APP Credora ou Proponente) antes das 10:00 do terceiro dia útil; (opção do cenário) A Credora envie a contraproposta dentro do período determinado e o cliente a rejeite ou aceite; Tanto a IF Credora quanto o cliente não tiveram ação, nesse caso, o status permanecerá como PENDING ao longo dos 3 dias úteis (Até as 10:00 do terceiro dia).
3. [ CANCELLED ] – Cliente cancela o pedido de portabilidade no APP da IF Proponente.
