## Fluxograma do processo de Portabilidade de Crédito no Open Finance

## Etapas

1. Contrato de crédito celebrado entre o Cliente e a IF Credora Original;
2. Cliente solicita a portabilidade de crédito junto a IF Proponente via Open Finance;
3. IF Proponente envia solicitação da proposta via API Padronizada;
4. Inicia-se o período da contraproposta, onde a IF Credora tem até 3 dias úteis para enviar uma proposta ao cliente ;
5. Dentro do período de 3 dias úteis a IF Credora deverá informar a retenção do cliente ou informar o saldo devedor com os dados para o TED, via API padronizada;
6. IF Proponente deverá efetivar a liquidação do contrato via TED na data do recebimento das informações;
7. Em até 2 dias úteis a IF Credora deverá confirmar o recebimento da TED e enviar a documentação atestando a efetivação da Portabilidade.
